#### 文字排版

`font-family`字体

```css
p{font-family:"Microsoft Yahei"}
```

`font-size`字号`color`颜色

```css
p{font-size:20px;color:red;}
```

`font-weight`设置文字的粗细

```css
p{font-weight:bold;}
```

`font-style`设置斜体文字

```css
p{{font-style:italic;}
```

`text-decoration:underline`下划线

```css
p {text-decoration: underline;}
```

`text-decoration:line-through`删除线

```css
p{text-decoration:line-through;}
```
